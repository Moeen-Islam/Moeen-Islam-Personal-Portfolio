# React + Tailwind Portfolio

A lightweight portfolio website built with Vite, React, Tailwind CSS, Framer Motion and React Icons.

## Run locally

```bash
npm install
npm run dev
```

Open the local URL shown in the terminal, usually http://localhost:5173

## Build

```bash
npm run build
npm run preview
```

## Customize

- Edit portfolio content in `src/data/portfolio.js`
- Main UI is in `src/App.jsx`
- Global styles are in `src/index.css`
- Replace images by changing image URLs or adding files inside `public/`

## Deploy to Vercel

1. Push the project to GitHub.
2. Import the repo in Vercel.
3. Build command: `npm run build`
4. Output directory: `dist`
